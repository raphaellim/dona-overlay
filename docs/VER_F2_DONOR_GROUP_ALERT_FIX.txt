Ver_f2 도네이터 기준 후원 ALERT 묶음 패치

요청 내용:
- 한 도네이터가 한 번에 여러 크리에이터에게 금액을 나눠 후원할 때 ALERT가 크리에이터 수만큼 반복 출력되는 문제 수정.
- ALERT 기준을 크리에이터 행 기준이 아니라 도네이터 기준으로 변경.

변경 파일:
- public/overlay.html

변경 동작:
- 같은 새 후원 조회 묶음 안에서 동일 도네이터명은 ALERT 1회로 합산 표시.
- 총금액은 도네이터 총합 기준 원단위로 표시.
- 크리에이터별 분배 금액은 만원단위로 표시.

예시:
- 도네이터A가 총 90,000원 후원
- 크리에이터 3명에게 30,000원씩 분배

기존:
- ALERT 3회 출력

변경 후:
- ALERT 1회 출력
- 도네이터A 90,000원
- 크리에이터1 3만 / 크리에이터2 3만 / 크리에이터3 3만

주의:
- 저장 데이터 구조는 유지합니다.
- 요약/상세/크리에이터별 합산/기존 후원 입력 기능은 그대로입니다.
- 무알림 처리된 후원은 기존처럼 ALERT에서 제외됩니다.
