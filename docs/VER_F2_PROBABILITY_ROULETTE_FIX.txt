Ver_f2 확률형 룰렛 추가 패치

적용 범위
- 기존 일반 룰렛 유지
- 룰렛 리스트 편집에 룰렛 방식 추가
  1) 일반 룰렛
  2) 확률형 룰렛
- PC 룰렛 / 모바일 룰렛 모두 확률형 생성 가능
- 서버 추첨 로직도 확률형 weightPercent 기준으로 당첨자 선택

확률형 입력 방식
- 한 줄에 하나씩 입력
- 이름 뒤에 확률 숫자 입력 가능
- % 글자는 넣어도 되고 안 넣어도 됨
- 확률 공란 가능

예시
ㅁㅁ 1.5
ㅇㅇ 3.3
ㅂㅂ 10
ㅎㅎ
ㅅㅅ

자동 계산 규칙
1. 공란 있음 + 입력 합계 100 이하
   - 입력된 확률을 먼저 고정
   - 남은 확률을 공란 항목에 균등 분배
   - 예: 1.5 + 3.3 + 10 = 14.8 / 남은 85.2 / 공란 2개 = 42.6씩

2. 공란 없음
   - 입력 숫자 비율대로 자동 보정해서 합계 100으로 저장
   - 예: 10 / 20 / 20 = 20 / 40 / 40

3. 공란 있음 + 입력 합계 100 초과
   - 입력된 숫자 비율대로 합계 100으로 자동 보정
   - 공란은 0 처리

기존 기능 유지
- 기존 일반 룰렛 리스트는 그대로 일반 룰렛으로 동작
- 자동 금액 룰렛 조건 유지
- 연속 룰렛 유지
- 오버레이 표시/사운드/RESULT 유지
