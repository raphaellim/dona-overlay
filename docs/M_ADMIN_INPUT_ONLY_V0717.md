# m_admin 입력 전용 개편 V0717

## 화면 구성
- 후원입력: 후원 금액과 크리에이터 분배를 한 화면에 통합
- 용돈
- 프리셋 수동 입력
- 미션타이머
- 룰렛 수동 실행/연속 실행/결과 숨김
- 공지 수정
- 펀딩 수동 입력

## 제거
- 미디어 입력 링크
- 모바일 설정/방송국 바로가기
- 최근 입력 목록
- 후원입력과 분배의 분리 탭

## 오류 수정
- 방송매니저가 미션타이머 노출 설정 저장 시 허용되지 않는 broadcastId가 전달되던 구조 제거
- 펀딩 수동 입력 시 원 단위 금액이 현재값에 그대로 더해지던 문제 수정
  - 10,000원당 펀딩 현재값 1 증가
- 후원 저장 시 입력 금액이 있는 크리에이터가 분배 대상으로 자동 체크되도록 개선
