# Donation Overlay Server - 상단 계좌/투네 + 하단 통합금액 버전

## 핵심 입력 방식

- 상단: 도네이터명, 계좌금액, 투네금액, 처리 선택
- 하단: 크리에이터별 금액 1칸만 입력
- 흡금/먹먹마는 크리에이터별 금액 기준으로 자동 계산
- 금액은 어떤 처리든 후원금액으로 합산

## 자동 배분 방식

하단 크리에이터 금액은 계좌/투네를 따로 입력하지 않기 때문에 저장 시 입력 순서대로 자동 배분됩니다.

예:
- 상단 계좌 50,000 / 투네 30,000
- 빵떠기 40,000
- 또영 40,000

저장 결과:
- 빵떠기: 계좌 40,000
- 또영: 계좌 10,000 + 투네 30,000

## 실행

```bash
npm install
npm start
```

## 접속

- 관리자 입력: `/admin.html`
- 합산 확인: `/summary.html`
- 방송컨트롤: `/control.html`
- OBS 화면: `/overlay.html`
